# FEATURE_012 - TUI 自动补全增强 - 人工测试指导

## 功能概述

**功能名称**: TUI 自动补全增强
**版本**: v0.5.0
**测试日期**: 2026-03-05
**测试人员**: [待填写]

**功能描述**:
对标 pi-mono 的自动补全系统，提升 KodaX 的命令行补全体验，包括：
- Fuzzy 模糊匹配算法
- `/skill:xxx` 技能补全
- 命令参数补全（如 `/mode`, `/thinking` 等）
- 自动触发补全（无需按 Tab）
- 下拉建议 UI

---

## 测试环境

### 前置条件
- Node.js >= 18.0.0
- 已运行 `npm install` 安装依赖
- 已运行 `npm run build` 构建项目

### 测试账号
- 无需特殊账号，使用本地环境

### 浏览器/环境要求
- 终端：Windows Terminal / macOS Terminal / iTerm2 / Linux Terminal
- 终端需支持 ANSI 颜色和 Unicode 字符
